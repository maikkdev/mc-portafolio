# 🟩 Minecraft Staff Portfolio Template

![Minecraft Banner](https://i.imgur.com/GrfHgVV.gif)

> **Plantilla gratuita** de portafolio para **Staff de Minecraft** (Admin, Mod, Helper, Head Staff)

---

## ✨ ¿Qué es esto?

Una plantilla moderna y con estilo 100% Minecraft para que **cualquier Staff** pueda crear su propio portafolio profesional de forma rápida y visual.

Diseñado con fuente oficial de Minecraft, partículas, efectos y colores del juego.

---

## 🚀 Cómo Usarlo (Muy Fácil)

1. **Haz clic en "Use this template"** (arriba a la derecha) o clona el repositorio.
2. Descarga o abre el archivo `index.html`
3. Edita los siguientes campos:

### Datos a personalizar:

- **Nombre** (`<h1>TuNick</h1>`)
- **Discord**
- **Email**
- **Enlaces** de GitHub, Discord, etc.
- **Información** de Sobre mí, Habilidades y Proyectos

---

## 🎨 Personalizar el Fondo

En el archivo `index.html`, busca esta línea y cambia el link:

```css
:root {
  --minecraft-bg: url('https://i.imgur.com/GrfHgVV.gif'); /* ← Cambia esta URL */
}
